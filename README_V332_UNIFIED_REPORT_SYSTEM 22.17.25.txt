V332 Unified Report System
- Adds one shared branded report header for all print/report screens.
- Uses assets/logo.png as the single Vardophase logo asset.
- Applies consistent white/rose-gold report styling to: Purchase Order print, Monthly Report, Project Report, Supplier Report, Supplier Credit Statement, Supplier Summary, Open Orders Report, Transparency Reports and GL Accounting Report.
- Does not change login, order save logic, GL item save logic, process/order status logic or supplier list data.
