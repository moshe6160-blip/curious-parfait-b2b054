V405 Mac Workspace Engine

Base: uploaded V403 build.

Added real workspace engine:
- Order / DN / Invoice / Deposit windows open with Gemini-style animation.
- Every entry window can be dragged freely on screen.
- Dock button saves the current draft window into the Mac-style dock.
- Restoring a docked window brings back the saved form values.
- You can dock one order and open another order.
- Dock can be hidden/shown when it interferes with work.

Untouched:
- Login
- Realtime
- Push
- Supabase
- Netlify functions
- Existing save logic
